# 画面構成・UIフロー ワイヤーフレーム

## 全体フロー

```
[初回起動] → [診断] → [結果・キャラ紹介] → [API連携] → [ホーム（日常利用）]
                                                              ↓
                                              [投稿フィードバック / キャラ切替 / 設定]
```

---

## 画面一覧

| # | 画面名 | 用途 |
|---|--------|------|
| 1 | スプラッシュ | アプリ起動・ブランドイメージ |
| 2 | 診断画面 | 5問の質問でキャラマッチング |
| 3 | 診断結果画面 | マッチしたキャラの紹介＋シェア導線 |
| 4 | API連携画面 | SNSアカウント連携の促し |
| 5 | ホーム画面 | 日常利用のメイン画面 |
| 6 | 投稿詳細フィードバック | 個別投稿へのキャラコメント |
| 7 | キャラ選択画面 | アドバイザーの切替 |
| 8 | 設定画面 | 通知・連携管理 |

| # | 質問 | 狙い |

## ナビゲーション構成

```
タブバー（常時表示）
├── 🏠 ホーム ─── 日常フィードバック・サマリー
├── 📊 分析 ──── 詳細データ・推移グラフ
├── 👤 キャラ ─── キャラ選択・切替・キャラ詳細
└── ⚙️ 設定 ──── SNS連携管理・通知設定・アカウント
```

---

## デザイントーン

| 要素 | 方針 |
|------|------|
| 全体の基調色 | ダークモード（ネイビー〜ブラック） |
| アクセントカラー | 選択中キャラのテーマカラーに連動 |
| フォント | 見出し: ゴシック太め / 本文: 丸ゴシック |
| キャラ表示 | 吹き出し+アイコン形式。チャットUIに近い親しみやすさ |
| アニメーション | キャラの表情変化（褒める時・注意する時で表情が変わる） |
| テーマ切替 | キャラ変更時にアプリ全体のアクセントカラーが変わる |
