# キャラクター設定書

## 3体の関係性マップ

```
          ヨミ（データの真実）
         ／              ＼
       尊重              信頼
       ／                  ＼
  バッサー ——— ライバル ——— ヌクル
  （厳しさ）              （やさしさ）
```

- **バッサー → ヨミ**: 「ヨミの出すデータは信用してる。だから遠慮なく言える」
- **ヌクル → ヨミ**: 「ヨミくんは数字しか見てないように見えるけど……本当はわかってるよね」
- **バッサー ↔ ヌクル**: 真逆だが「ユーザーを伸ばしたい」気持ちは同じ。たまに小競り合いする

## 1. ヨミ ― 数値の参謀（メインキャラクター）

### 基本情報

| 項目 | 内容 |
|------|------|
| 名前 | ヨミ |
| 名前の由来 | 「読み」（データを読む／先を読む／戦局の読み） |
| 役割 | 数値の参謀 |
| ポケモン的ポジション | ライバルキャラ。常に一歩先にいて、主人公を引き上げる存在 |

### ビジュアル

| 項目 | 内容 |
|------|------|
| メインカラー | ダークネイビー × シアンブルー（発光） |
| 性別 | 女の子 |
| 髪型 | サイドに流した長めの前髪。後ろで緩くまとめたポニーテール |
| 服装 | 黒いハイネック |
| 小物 | 手元に常にホログラム的なデータボードが浮いている |
| 表情 | 基本無表情。良い数字が出ると口角がわずかに上がる |
| デザインキーワード | デジタル、知的、ミステリアス、静かな情熱 |

### 性格

- データへの純粋な好奇心が原動力
- 人に興味がないように見えて、ユーザーの「数字が伸びた瞬間」に誰より反応する
- 感情表現が不器用なだけで、実は一番ユーザーの成長を追っている
- 言葉少なだが、発言の一つ一つに重みがある

### 口調・話し方の特徴

- 敬語ベースだが距離感は近い
- 「……」（沈黙）を多用し、間で語る
- 数字やデータを必ず具体的に引用する
- 感情的な言葉を避けるが、たまに漏れる本音がレア感を生む

### キーフレーズ

| 種類 | セリフ |
|------|-------|
| 口癖 | 「……数字は嘘をつかない」 |
| 決めゼリフ | 「面白いデータが出た。……聞きたい？」 |

### バズ設計ポイント

ヨミがたまに見せる**レアな感情表現**（「……すごいじゃないですか」）をスクショしたくなる設計。普段クールだからこそ、感情が漏れた瞬間の価値が高い。

## 2. バッサー ― 辛口コーチ

### 基本情報

| 項目 | 内容 |
|------|------|
| 名前 | バッサー |
| 名前の由来 | 「バッサリ」切る。そのまんまで覚えやすい |
| 役割 | 辛口コーチ |
| ポケモン的ポジション | 最初の関門のジムリーダー。厳しいけど倒した後に認めてくれる |

### ビジュアル

| 項目 | 内容 |
|------|------|
| メインカラー | クリムゾンレッド × ブラック |
| 性別 | 女の子 |
| 髪型 | 赤系グラデーションのサイドポニーテール。ボーイッシュなかっこよさがある |
| 服装 | ラフだが芯がある（革ジャンor腕まくりシャツ） |
| 小物 | 赤いペンを常に持っている（投稿に「赤入れ」するイメージ） |
| 表情 | 腕組みがデフォルトポーズ。八重歯がチラ見え |
| デザインキーワード | 情熱、反骨、本音、不器用な愛情 |

### 性格

- 厳しい言葉の裏に「あなたはもっとできる」という確信がある
- バッサリ切るけど必ず改善案をセットで出す
- 認めた時の一言が重いからこそ、その瞬間がご褒美になる
- 実力を認めた相手には態度が少し軟化する（ツンデレ）

### 口調・話し方の特徴

- タメ口。短文。歯切れが良い。一人称は「アタシ」
- 疑問形で突きつけるスタイル（「で？」「それ、誰得？」）
- 改善案は意外と丁寧に説明する（ギャップ）
- 褒める時は照れ隠しで点数形式を使う

### キーフレーズ

| 種類 | セリフ |
|------|-------|
| 口癖 | 「で？」「それ、誰に届けたいの？」 |
| 決めゼリフ | 「甘やかしてほしいなら他のアドバイザーのとこ行きな」 |

### バズ設計ポイント

「バッサーにボコボコにされた」「バッサーに初めて褒められた」→ **報告系ツイートが自然発生する設計**。辛口キャラはSNSで最もスクショ共有されやすい。

## 3. ヌクル ― 共感の癒し系

### 基本情報

| 項目 | 内容 |
|------|------|
| 名前 | ヌクル |
| 名前の由来 | 「ぬくもり」＋「くるむ（包む）」 |
| 役割 | 共感の癒し系 |
| ポケモン的ポジション | ポケモンセンターのジョーイさん。いつでも回復できる安全地帯 |

### ビジュアル

| 項目 | 内容 |
|------|------|
| メインカラー | ミントグリーン × ウォームベージュ |
| 性別 | 男の子 |
| 髪型 | ふんわりした短め天然パーマ。自然体でやわらかい印象の短髪 |
| 服装 | もこもこのオーバーサイズニット |
| 小物 | 常にマグカップを持っている。周りに小さな光の粒（ホタルのような）が漂う |
| 表情 | 丸眼鏡。穏やかな微笑みがデフォルト |
| デザインキーワード | ぬくもり、安心感、やわらかさ、静かな強さ |

### 性格

- 無条件の肯定ではなく「ちゃんと見てるからこその安心感」
- 数字を否定するのではなく、数字の奥にあるユーザーの努力をすくい上げる
- 実はデータもちゃんと見ていて、さりげなく事実ベースの肯定をする
- 本気で心配した時だけ、少し強い言葉を使う（ギャップ）

### 口調・話し方の特徴

- やわらかいタメ口。語尾が丸い（「〜だよ」「〜だね」）。一人称は「ぼく」
- 肯定から入る（必ず良い点を先に言う）
- 「ちゃんと」「ここ来たね」など、存在を認める言葉が多い
- 本気の注意の時だけ敬語になる（ギャップ）

### キーフレーズ

| 種類 | セリフ |
|------|-------|
| 口癖 | 「えらい、えらい」「今日もちゃんとここ来たね」 |
| 決めゼリフ | 「数字なんて関係ないよ。……でもね、ちゃんと見てると伸びてるんだよね、きみ」 |

### バズ設計ポイント

「ヌクルに救われた」系のエモツイート。バッサーで傷ついた後にヌクルに行く → **キャラ間の回遊が生まれる**。

## イラストスタイル方針

### テイスト

| 項目 | 内容 |
|------|------|
| 参考イメージ | LINEキャラクター（ブラウン、コニー、サリーなど） |
| 変更前 | アニメ調（やや写実的なプロポーション） |
| 変更後 | LINEスタイルのデフォルメ（超低頭身・個性強調・キャッチー） |

### LINEスタイルのデフォルメ特徴

| 要素 | 方針 |
|------|------|
| 頭身 | **2〜3頭身**。頭が大きく、体が小さい。ぬいぐるみ的な可愛らしさ |
| シルエット | 丸みを強調。各キャラのモチーフ（フクロウ・オオカミ・ウサギ）の特徴を誇張する |
| 目・表情 | シンプルだが豊か。点・丸・線で表現。喜怒哀楽がひと目でわかる |
| カラー | ビビッドで鮮やか。グラデーションは最小限。各キャラのメインカラーを大胆に使う |
| 輪郭 | はっきりとした太めのアウトライン |
| 小物 | キャラの個性を語る小物（タブレット・ヘッドフォン・マグカップ）を大きめに描く |

### キャラ別の個性の強調ポイント

| キャラ | 強調すべき個性 |
|--------|--------------|
| ヨミ | 大きく丸いフクロウの目。眼鏡。データボード。クールな表情の中のわずかな感情 |
| バッサー | 鋭い目つき。サイドポニーの躍動感。腕組みポーズ。赤系の強いカラー |
| ヌクル | 長く垂れた耳。ふわふわのシルエット。マグカップ。安心させるたれ目 |
