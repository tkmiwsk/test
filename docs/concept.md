# SNS投稿パーソナルアドバイザー ― 企画概要

## コンセプト

**「発信をもっと楽しくする、あなた専属のSNSアドバイザー」**

SNSの投稿データをもとに、個性豊かなキャラクターがフィードバックをくれるパーソナルアドバイザーサービス。
あすけん（ダイエット）のSNS版として、エンタメ性とキャラクターの魅力でAPI連携のハードルを下げる。

## ベンチマーク

- **あすけん** https://www.asken.jp/ ― キャラによるアドバイス体験のUI/UX
- **ポケットモンスター** ― キャラクターデザインの哲学（一目で性格がわかる・名前が属性を暗示・決めゼリフ）

## ターゲット

インフルエンサー（SNS API連携をしてもらいたいユーザー）

## 解決する課題

| 課題 | アプローチ |
|------|-----------|
| API連携の動機付けが弱い | キャラクターの魅力で「使いたい」を生む |
| 機能的メリットだけでは不十分 | エンタメ性・雰囲気・コンセプトで訴求 |
| SNS運用に明確な1つの課題がない | 複数キャラで多角的にアプローチ |
| 継続利用のモチベーション | キャラへの愛着・キャラ間の回遊で継続 |

## MVP構成

3体のアドバイザーキャラクターでローンチ。

| 枠 | キャラ名 | 役割 | 選定理由 |
|----|---------|------|---------|
| メイン | ヨミ | 数値の参謀 | クール・知的。軸となる信頼感 |
| 2体目 | バッサー | 辛口コーチ | 「roastされた」スクショ拡散 → バズ製造機 |
| 3体目 | ヌクル | 共感の癒し系 | 辛口の対極。感情的な引力で継続率を支える |

## バズ導線設計

```
初回診断結果のシェア ─────── バズ導線①
日々のフィードバックのスクショ共有 ── バズ導線②
キャラ切替の報告ツイート ────── バズ導線③
キャラ間の掛け合い ─────── コンテンツとしての拡散
```
